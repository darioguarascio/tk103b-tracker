Lightweight GPS tracking for TK103B devices

Self-hosted dashboard with live map, trip replay, event filtering, and a TCP ingest server that speaks the TK103B protocol. Store positions in PostGIS, view them in a modern web UI.

Get started View source

What is it?

TK103B Tracker is an open-source stack for cheap Chinese GPS trackers using the TK103B text protocol. It replaces heavy CMS setups with a focused map interface: see where your vehicle is now, replay where it was, and inspect alarms and ignition events on a timeline.

📡 Ingest

TCP server on port 9000 receives tracker packets, parses coordinates and events, writes to PostGIS.

🗺️ Web UI

React + Leaflet map with live SSE updates, replay mode, and per-event-type filtering.

🗄️ PostGIS

Spatial queries for tracks, latest positions, and event history. Bring your own database.
