Toolbar

  • Tracker — select which IMEI to view
  • From / To — datetime range for replay (disabled in Live mode)
  • Event chips — toggle which event types to load and show on the map
  • Reload — fetch track + events for the selected range
  • Replay / Live — switch between historical playback and real-time SSE stream
  • Play / Speed — animate replay at 1×–16× (replay mode only)

Map

  • Blue polyline — move track; dashed overlay for depth
  • Blue dot — current position (replay frame or live update)
  • Coloured dots — events (alarms, acc on/off, etc.); click for popup with time and speed
  • Auto fit-bounds when loading a track; live mode pans to follow the vehicle

Sidebar & status bar

  • Event list — click an event in replay mode to jump the timeline to that moment
  • Status bar — current time, speed, heading, event type, point count, last seen
  • Timeline slider — scrub through replay frames manually

Login screen

When AUTH_PASSWORD is set, a simple password gate appears before the dashboard. Sessions last 7 days via signed cookie.
