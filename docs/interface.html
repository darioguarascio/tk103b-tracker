Toolbar

  • Tracker — select which IMEI to view
  • Mode — Replay (historical data) or Live (real-time SSE stream)
  • Range — date presets (last day, week, month, …) or Custom with manual from/to
  • From / To — datetime range for replay (disabled in Live mode)
  • Event chips — toggle which event types to load and show on the map
  • Reload — fetch track + events for the selected range

On desktop the toolbar is always visible. On mobile it moves into a slide-out Controls drawer (☰ menu).

Playback bar

In replay mode, a video-player-style control row appears in the status bar (not in the toolbar):

  • Play / Pause — animate through move points
  • Speed — 1×, 2×, 4×, 8×, or 16×
  • Timeline slider — scrub to any frame; pauses playback
  • Frame counter — current position / total points

Playback controls are hidden in Live mode.

Map

  • Blue polyline — move track; dashed overlay for depth
  • Blue dot — current position (replay frame or live update)
  • Coloured dots — events (alarms, acc on/off, etc.); click for popup with time and speed (at zoom ≥14)
  • Auto fit-bounds when loading a track; live mode pans to follow the vehicle

Sidebar & status bar

  • Event list — non-move events only; click in replay mode to jump the timeline to that moment
  • Status bar — current time, speed, heading, event type, window point count, last seen
  • Playback bar — play, speed, and timeline scrubber (replay only; see above)

Mobile

  • Header — ☰ opens Controls drawer; Events opens a bottom sheet
  • Map — full screen; sidebar hidden
  • Controls drawer — same fields as the desktop toolbar, stacked in a column
  • Playback bar — pinned to the footer, full width above position stats

Login screen

When AUTH_PASSWORD is set, a simple password gate appears before the dashboard. Sessions last 7 days via signed cookie.
