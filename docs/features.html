Features

Everything you need to track one or more TK103B devices without vendor lock-in.

●

Live tracking

Server-Sent Events stream new positions as they arrive. The map follows the vehicle and appends to the track in real time.

▶

Trip replay

Scrub through historical move points with a video-style control bar: play/pause, speed (1×–16×), and timeline slider. Hidden in live mode.

⚡

Event types

Filter and display alarms and state changes: acc on/off, sensor alarm, acc alarm, tracker heartbeat, AC alarm.

🛰

Multi-tracker

Devices auto-register by IMEI on first packet. Switch between trackers from a dropdown; each has independent history.

📱

Mobile layout

Map-first on small screens: hamburger menu for controls, bottom sheet for events, playback bar pinned to the footer.

📅

Date presets

Quick ranges (last day, 3 days, week, 2 weeks, month) or custom from/to datetime pickers.

🛡

Smart filtering

GPS jump filter drops implausible teleports; stationary move points (0 km/h) are excluded from tracks.

🔒

Optional password auth

Set AUTH_PASSWORD to protect the UI. Signed HTTP-only cookies; leave empty for open access on trusted networks.

🐳

Docker-ready

Two images on Docker Hub — app and ingest. Sample compose file in the repo; bring your own PostGIS instance.

API endpoints

The Express API serves the React UI and exposes JSON endpoints:

EndpointPurpose
GET /api/trackersList devices with last known position
GET /api/trackPosition history for replay
GET /api/eventsNon-move events in a time range
GET /api/liveSSE stream of new positions
GET /api/rangeEarliest/latest timestamps in DB
GET /api/healthHealth check (DB connectivity)
